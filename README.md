# _TRAKTION_


#### Traktion is a habit forming, notification and tracking app, 01-14-2021

#### By _**Jonathan Quesada, Jena Boehm, Rosalie Lee, Perry Sittser, Katie Berry, Andrew Bray**_ 


## Description

Traktion is an entirely backend SMS-based app that helps users form new habits. The user sets up their reminders, and upon receiving the reminders they respond with a photo of them engaging in their habit of choice. Traktion verifies the userâ€™s face and habit in the photo and then responds with reinforcement and confirmation of completion.
